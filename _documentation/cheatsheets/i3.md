# i3 Cheat Sheet

## Additional Softwares
Terminal : `kitty`
File Manager : `thunar`
Web Browser : `vivaldi`
Status Bar : `polybar`
Wallpaper Engine : `feh`
Screen Locker : `cinnamon-screensaver`
App launcher : `rofi`
Power Menu : `rofi` + `rofi-power-menu`

## Key Binds
|         Bind          | Description                                           | Command                                                                                                                                                |
|:---------------------:|:------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------|
|     Super + PgUp      | Brighten Screen by 10%                                | brightnessctl s +10%                                                                                                                                   |
|     Super + PgDn      | Dim Screen by 10%                                     | brightnessctl s 10%-                                                                                                                                   |
|    Super + Return     | Start default i3 terminal (generally xterm or uxterm) | i3-sensible-terminal                                                                                                                                   |
|       Super + Q       | Start Kitty terminal                                  | kitty *using $terminal variable*                                                                                                                       |
|       Super + L       | Lock Screen                                           | cinnamon-screensaver-command --lock                                                                                                                    |
|       Super + E       | Open File Manager                                     | thunar *via $fileManager variable*                                                                                                                     |
|       Super + B       | Open Web Browser                                      | vivaldi *via $webBrowser variable*                                                                                                                     |
|       Super + C       | Close active window                                   | kill                                                                                                                                                   |
|       Super + R       | Open App Launcher                                     | rofi -show drun -show-icons                                                                                                                            |
| Super + Shift + Down  | Play / Pause Media                                    | playerctl play-pause                                                                                                                                   |
| Super + Shift + Left  | Previous Media                                        | playerctl previous                                                                                                                                     |
| Super + Shift + Right | Next Media                                            | playerctl next                                                                                                                                         |
|     Super + Left      | Move focus to the left                                | left                                                                                                                                                   |
|     Super + Down      | Move focus down                                       | down                                                                                                                                                   |
|     Super + Right     | Move focus to the right                               | right                                                                                                                                                  |
|      Super + Up       | Move focus up                                         | up                                                                                                                                                     |
|  Super + Ctrl + Left  | Move window to the left                               | move left                                                                                                                                              |
|  Super + Ctrl + Down  | Move window down                                      | move down                                                                                                                                              |
| Super + Ctrl + Right  | Move window to the right                              | move right                                                                                                                                             |
|   Super + Ctrl + Up   | Move window up                                        | move  up                                                                                                                                               |
|       Super + F       | Toggle current window fullscreen                      | fullscreen toggle                                                                                                                                      |
|       Super + V       | Toggle current window floating                        | floating toggle                                                                                                                                        |
|   Ctrl + Alt + Left   | Go to previous opened workspace                       | workspace prev                                                                                                                                         |
|  Ctrl + Alt + Right   | Go to next opened workspace                           | workspace next                                                                                                                                         |
|      Super + {n}      | Go to workspace number {n}                            | workspace number $ws{n}                                                                                                                                |
|  Super + Ctrl + {n}   | Move current window to workspace {n}                  | move container to workspace number $ws{n}                                                                                                              |
|      Super + F5       | Reload i3                                             | reload                                                                                                                                                 |
|  Super + Shift + F5   | Restart i3                                            | restart                                                                                                                                                |
|       Super + M       | Display power menu                                    | rofi -show power-menu -modi power-menu:~/.local/bin/rofi-power-menu -theme gruvbox-dark-hard"                                                          |
|   Super + Shift + M   | Exit i3                                               | i3-nagbar -t warning -m 'You pressed the exit shortcut. Do you really want to exit i3? This will end your X session.' -B 'Yes, exit i3' 'i3-msg exit'" |
|   Super + Shift + R   | Enter resize mode                                     | mode "resize"                                                                                                                                          |
